# Support

This organization contains **private research repositories**.
There is no public support channel.

## For Internal Contributors

- **Task backlog:** check `TASKS.md` in the relevant repo
- **Session context:** check `DEVLOG.md` in `ip-vault`
- **Architecture questions:** check `docs/architecture/` in `ip-vault`
- **Issues:** open an issue in the relevant repository using the provided templates
